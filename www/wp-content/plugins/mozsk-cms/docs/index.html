Mozilla.sk CMS Plugin documentation

Mozilla.sk CMS Plugin is plugin for WordPress which can:

  • Manage downloads of Mozilla products.
  • Manage links to articles about Mozilla.

How to install

  • Copy directory mozsk-cms to directory plugins in your WordPress installation.
  • Set your locale in file config.php.

How to use plugin

Administration

Visit WordPress administration and have a look into section Plugins. You can find two new pages there (Products and Articles about Mozilla). I think it's simple to use so no special documentation is needed. Maybe short note to field Text ID in form Add product. In this field fill short product name like firefox, thunderbird, seamonkey etc. If you use seamonkey (all small chars) for SeaMonkey than function Autofill will set link to release notes correctly.

Template

How to print current application version

echo get_product_version('text id');

How to print link to release notes for latest application version

echo get_product_rn('text id');

How to print link for downloading latest application version

echo get_product_link('text id', 'os');

In paramether os you can use values: win, lin, mac or port

Content

You can print informations from plugin in posts/pages via Shortcode API.

How to print current application version

[moz-download-version app="text id"]

How to print link to release notes for latest application version

[moz-download-rn app="text id"]

How to print link for downloading latest application version

[moz-download-url app="text id " platform="os"]

In paramether os you can use values: win, lin, mac or port
