Vítá vás poštovní program aplikace Mozilla

Program pro zpracování elektronické pošty a pro čtení příspěvků z diskusních skupin aplikace Mozilla je klientem, kterého lze provozovat na více platformách, má podporu pro protokoly SMTP, POP3, IMAP, NNTP a LDAP. Rozhraní aplikace je rozmístěno do třech panelů, nabízí přehledné vyhledávání a filtrování zpráv, podporuje vlastní databázi kontaktů a lze použít více identit při odesílání.

Narozdíl od předchozích verzí aplikace Netscape podporuje Mozilla vícenásobné POP účty. K dalším vlastnostem patří podpora více SMTP serverů a historie databáze kontaktů.

Vítáme ohlasy či hlášení o chybách

Předtím, než pošlete hlášení o chybě, si prosím přečtěte poznámky k verzi. Je možné, že se již o daném problému ví.

Tato verze byla uvolněna pro účely testování, takže můžete také pomoci tím, že budete program užívat a případné problémy lze nahlásit prostřednictvím stránky Bugzilla. O tom, jak lze testovat program pro čtení pošty a diskusních skupin se můžete dočíst v dokumentu Začínáme.

Pokud hledáte samostatného poštovního klienta, vyzkoušejte Mozilla Thunderbird.
