Open Source and Mozilla

– for the good of the code

Mozilla je otevřený [en] a svobodný software [en] – každý jednotlivec nebo společnost může:

  • spouštět program za jakýmkoli účelem;
  • studovat jak program funguje a měnit ho podle svých potřeb;
  • distribuovat program jak chce;
  • vylepšit program a distribuovat ho jako alternativní verzi.

Veškeré zdrojové kódy Mozilly jsou k dispozici pod Mozilla a Netscape Public Licenses [en], které byly přijaty jako free software licenses [en] společností Free Software Foundation [en], která zaštiťuje veškerý svět otevřeného software.

Duch MPL [en] říká, že je vám dovoleno používat kód Mozilly ve vašich aplikacích a produktech – včetně proprietárních produktů –, které nabízíte, pokud zveřejníte všechny změny původního kódu Mozilly, které jste sami provedli.

Se svobodným softwarem nejsou vaše obchody vázány [en] na produkt žádné jiné společnosti. Jste svobodní v kontrole své budoucnosti.
